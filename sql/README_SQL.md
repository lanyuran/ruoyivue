# SQL 文件说明

本目录仅保留**可部署的最终 SQL**，包含基础系统数据与业务表结构，不含业务示例数据与完整快照。

## 文件用途
- `sql/0_整合脚本.sql`
  - 依次拼接 `1_创建数据库.sql` → `2_基础运行数据.sql` → `3_业务.sql`。
- `sql/1_创建数据库.sql`
  - 创建数据库 `fl` 与本地用户 `fluser`，授予 `fl.*` 权限。
- `sql/2_基础运行数据.sql`
  - RuoYi 基础表结构与初始化数据（系统/权限/菜单/角色/字典等）。
  - 包含 Quartz 调度表结构与系统功能所需权限扩展。
- `sql/3_业务.sql`
  - 业务表结构（患者信息、就诊信息、问卷），**不包含示例数据**。

## 推荐执行方式
- **完整部署（推荐）**：执行 `0_整合脚本.sql`
- **分步执行**：`1_创建数据库.sql` → `2_基础运行数据.sql` → `3_业务.sql`

> 执行前请确认使用目标数据库（建议 `USE fl;`）。
